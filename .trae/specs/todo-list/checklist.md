# 可拖动待办清单 (Tolist) - 验证清单

## 核心功能验证
- [ ] 待办面板可以在屏幕范围内自由拖动
- [ ] 拖动时不会拖出屏幕外
- [ ] 拖动到左/右/顶部边缘时自动吸附
- [ ] 贴边后鼠标离开3秒自动隐藏，只显示小把手
- [ ] 隐藏状态下鼠标悬浮到小把手可以展开面板
- [ ] 刷新页面后面板自动恢复到上次的位置和状态

## 待办管理验证
- [ ] 可以正常添加新待办
- [ ] 可以删除待办
- [ ] 双击待办项可以编辑内容
- [ ] 可以标记待办完成/未完成
- [ ] 完成的待办有明显的视觉标记（如删除线、灰色）
- [ ] 可以设置待办优先级（高/中/低）
- [ ] 不同优先级有不同的颜色标记
- [ ] 可以按分类筛选待办（工作/生活/学习）
- [ ] 可以设置待办截止时间
- [ ] 超时的待办自动标红提醒
- [ ] 待办创建时间正确显示
- [ ] 可以拖拽待办项调整顺序
- [ ] 一键清空已完成功能正常工作
- [ ] 一键全选/反选功能正常工作

## 数据持久化验证
- [ ] 添加待办后刷新页面，数据保持不变
- [ ] 编辑待办后刷新页面，修改生效
- [ ] 删除待办后刷新页面，删除生效
- [ ] 标记完成后刷新页面，状态保持
- [ ] 待办顺序调整后刷新页面，顺序保持

## 主题适配验证
- [ ] 暗黑模式下，待办面板样式正确
- [ ] 浅色模式下，待办面板样式正确
- [ ] 切换主题时，待办面板样式同步更新

## 快捷键验证
- [ ] 按 Enter 键可以快速添加待办
- [ ] 按 Ctrl/Command + D 可以标记待办完成
- [ ] 按 Esc 键可以取消编辑或收起面板

## 交互体验验证
- [ ] 可以展开/折叠面板
- [ ] 折叠后只显示标题栏
- [ ] 有未完成待办时，隐藏状态下显示小红点
- [ ] 所有待办完成后，小红点消失
- [ ] 待办添加/删除时有平滑的动画效果
- [ ] 贴边隐藏/展开有平滑的动画效果

## 数据导出验证
- [ ] 可以导出待办为文本文件
- [ ] 导出的文件内容正确

## 代码质量验证
- [ ] 类型定义完整、清晰
- [ ] 代码结构清晰，符合现有风格
- [ ] 使用现有的 SmartStorageManager 进行数据持久化
- [ ] 使用现有的 ThemeContext 进行主题适配
- [ ] 使用 Ant Design 组件库
- [ ] 代码有适当的注释（如果需要）
